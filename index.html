URL → Notion 전송

GitHub Pages에서 URL을 입력하면 Vercel 서버리스 API를 통해 노션 DB로 저장됩니다.

여러 항목을 선택하세요(⌘/Ctrl 클릭). 필요하면 위 입력창으로 새 항목을 추가할 수 있습니다. 노션 DB의 Category multi-select로 저장됩니다.

README: 설정 가이드

현재 API 엔드포인트는 https://urln.vercel.app/api/submit 으로 고정되어 있습니다.

  • 폼의 카테고리는 다중 선택(select multiple) 방식입니다.
  • 선택된 값들은 노션 DB의 Category multi-select 속성으로 저장됩니다.
