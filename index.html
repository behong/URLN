URL → Notion 전송

GitHub Pages에서 URL을 입력하면 Vercel 서버리스 API를 통해 노션 DB로 저장됩니다.

끝에 /api/submit 없이, 도메인만 적습니다.

README: 설정 가이드

  1. 노션 DB 준비
    • 노션에서 데이터베이스(보드/테이블) 하나 생성
    • 필수 속성:
      • Title (제목) – title 타입
      • URL – url 타입
      • Notes – rich_text 타입 (선택)
    • 주소창에서 데이터베이스 ID 복사 (페이지 URL 중 32자-하이픈 제거 형태도 허용)
    • 노션 ▶︎ Settings & Members → Connections에서 Integrations 만들고, 해당 DB에 연결(Share)로 권한 부여
  2. Vercel 서버리스 API
    • 새 프로젝트 생성 후, 아래 api/submit.ts 파일 추가
    • 프로젝트 환경변수 설정:
      • NOTION_TOKEN = 시크릿 통합(Integration) 토큰
      • NOTION_DATABASE_ID = 위에서 복사한 DB ID
  3. GitHub Pages 프런트
    • 이 index.html을 리포지토리에 올리고 Pages 활성화
    • 페이지에서 상단의 API 엔드포인트에 본인 Vercel 도메인(예: https://your-app.vercel.app) 입력

프런트엔드 JS (이 페이지 하단 스크립트)

폼 제출 시 API로 { url, title, notes }를 POST 합니다.
